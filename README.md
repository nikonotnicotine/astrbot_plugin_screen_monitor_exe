# 我会一直视监你的，永远 (Screen Spy Server)

这是为 [AstrBot 屏幕查岗插件](https://github.com/nikonotnicotine/astrobot_plugin_screen_spy) 设计的本地截图服务端程序。

下载并运行后，它会作为一个隐藏在系统托盘的小程序，在后台默默等待 AI 的“查岗”指令。

## 🚀 方式一：直接运行 (推荐)

直接下载本项目中的 **`我会一直视监你的，永远.exe`** 文件，然后双击运行即可。

程序会自动隐藏到你电脑右下角的系统托盘，无需任何额外操作。

## 🛠️ 方式二：使用 Python 脚本

如果 `.exe` 在你的电脑上无法运行，你可以尝试使用 Python 脚本。

### 1. 安装依赖

首先，确保你已经安装了 Python。然后打开命令行 (CMD)，输入以下命令安装所需库：

```bash
pip install flask mss pillow pystray
```

### 2. 运行脚本

1.  下载本项目中的 **`server.py`** 文件。
2.  在 `server.py` 文件所在的文件夹路径栏输入 `cmd` 并回车，打开命令行。
3.  在命令行中输入以下命令并运行：

    ```bash
    python server.py
    ```
    > 注意：通过这种方式运行时，**请不要关闭弹出的黑色命令行窗口**，否则程序会退出。

## ✨ 程序特性

- **后台运行**: `.exe` 版本可完全在后台运行，无黑框。
- **自定义图标**: 在程序同目录下放置 `icon.ico` 文件即可自动应用。
- **运行日志**: 双击托盘图标或右键菜单可随时查看运行状态和AI查岗记录。
- **开机自启**: 右键菜单可方便地开启或关闭开机自启功能。
